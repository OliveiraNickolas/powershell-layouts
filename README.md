# SnapLayout v4

Gestor de janelas com Spaces e Atalhos para Windows PowerShell.

## Funcionalidades

- **Spaces**: Organize janelas em áreas definidas
- **Atalhos por Space**: Atalhos globais para encaixar janelas em espaços específicos
- **Layouts Salvos**: Salve e carregue configurações de layouts
- **Snapshot**: Capture o layout atual das janelas na tela

## Como Usar

1. Execute `layouts.ps1` no PowerShell
2. Use "Capturar Layout" para salvar a disposição atual das janelas
3. Configure atalhos para cada space para encaixe rápido de janelas
4. Aplique layouts salvos com um clique ou atalho global

## Requisitos

- Windows PowerShell 5.1+ ou PowerShell 7+
- Sem dependências externas
